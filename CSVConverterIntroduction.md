## Features available in version 0.1 ##

Huge files

Kinds of mapping managed:
  * One to one,
  * One to many,
  * Many to one,
  * Many to many.

Error management :
  * Error file in CSV,
  * Error reason file in TXT format.

Header management (could be removed).
